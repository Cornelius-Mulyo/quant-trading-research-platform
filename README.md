# Quantitative Trading Research Platform

A quantitative research platform for developing, testing, and optimizing algorithmic trading strategies.

## Features

- Market data ingestion
- Statistical arbitrage research
- Backtesting engine
- Portfolio optimization
- Risk analytics
- Interactive dashboard

## Technologies

- Python
- PostgreSQL
- Pandas
- NumPy
- SQLAlchemy
- Statsmodels
- CVXPY
- Streamlit
